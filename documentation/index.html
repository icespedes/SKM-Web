Installing Template.

  1. After unzip the download pack, you'll found a Template Folder with all the files.
  2. You can view this Template in any browser, you can display or edit the Template without an internet connection.
  3. The only section that will not work is the Contact Section which contains PHP code and requires a server environment for send messages.
  4. Now go to your cpanel or open your FTP Client (like Filezilla) and upload the content of the Template on your server root.
  5. Once the files are done uploading go to www.yourdomainname.com/index.html

Setup Twitter Feed

New Twitter API requires oAuth Token Key. Here i will show you how to setup you twitter account

  1. Go to https://apps.twitter.com/, Sign in with your twitter account.
  2. After log in you see a button named Create New App, see the image below
  3. After click on create new app button, you see window like this, see the image below. Fill out this form and click on Create your Twitter application. Okay your twitter application is created.
  4. After creating your twitter application you see a dashboard of your application, see the image below. Now click on API Keys
  5. Collect your API key and API secret. Now go to your theme's root directory and find the config.php . It's located at main / HTML / api / config.php.
    Edit config.php with notepad++
  6. Now replace CONSUMER_KEY_HERE with your API key
    and replace CONSUMER_SECRET_HERE with your API Secret Key
    Now Save config.php and go to twitter application dashboard and scroll down and find Create My Access Token, now click on this button, see the image below

    If you see this message

    click on refresh
  7. After refreshing you fing your Access token and Access token secret ,see the image below

  8. Now edit config.php again and replace ACCESS_TOKEN_HERE with your access token key and
    replace ACCESS_SECRET_HERE with your access token secret key and save it.

    9. Okay you're done!! Cheers!!! Now your twitter feed is live on your site :D

    Contact Form

	                    $name = $_REQUEST['name'];

	                    $subject = $_REQUEST['subject'];

	                    $fields = array();
	                    $fields["name"] = $_REQUEST['name'];
	                    $fields["email"] = $_REQUEST['email'];
	                    $fields["subject"] = $_REQUEST['subject'];
	                    $fields["message"] = $_REQUEST['message'];

	                    $body = "Message From Site name:\n\n

	                    $body .= 'Name : '.$fields['name']. '

	                    $body .= 'Email : '.$fields['email']. '

    ';
